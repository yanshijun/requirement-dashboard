# 需求池看板 · 功能清单

## 项目概述
客服GPT SAE版需求池看板，基于飞书多维表格存储数据，单页应用（纯 HTML + JS），支持需求全生命周期管理。

---

## 一、需求列表（📋）

### 1.1 状态统计卡片
- 展示全部需求、紧急、待开始、开发中、联调中、测试中、已上线、已暂缓共 8 个统计数字
- 点击任意卡片可跳转到开发看板并自动筛选对应状态
- 点击"全部需求"跳转看板不带筛选条件

### 1.2 筛选工具栏
- 按所属计划筛选
- 按需求类型筛选（用户需求 / 研发需求）
- 按优先级筛选（紧急🔴 / 高🟠 / 中🟡 / 低🟢）
- 按开发状态筛选（待开始 / 开发中 / 联调中 / 测试中 / 已上线 / 已暂缓）
- 按认领人筛选
- 关键词搜索（需求名称 + 描述模糊匹配）

### 1.3 需求表格
- 展示字段：编号、需求名称、类型、优先级、所属计划、需求描述（截断80字）、认领人、状态、进度条、预计上线、提出时间、操作
- 优先级列支持点击排序（升序 / 降序 / 默认）
- 默认按提出时间降序排列（最新在前）
- 分页展示，每页 20 条，支持智能页码（最多显示7个，超出显示省略号）
- 翻页后自动滚动到顶部

### 1.4 认领管理
- 待认领需求显示"认领"按钮
- 已认领需求显示"换人"和"释放"按钮
- 认领弹窗支持多选认领人（复选框）
- 支持在认领弹窗内新增成员
- 支持从成员列表移除成员（软删除，存 localStorage）
- 支持单独移除某一位认领人（×号）

---

## 二、开发看板（🗂）

- 按状态分列展示：待开始 / 开发中 / 联调中 / 测试中 / 已上线 / 已暂缓
- 每列显示需求卡片，包含：需求名称、类型标签、优先级标签、认领人、进度条
- 点击卡片打开需求详情
- 从需求列表状态统计卡片跳转时自动筛选对应列

---

## 三、人员任务（👤）

- 按认领人分组展示任务卡片
- 每张卡片显示：姓名、平均进度条、已上线/进行中/待开始数量统计、任务列表
- 任务列表支持滚动查看（最大高度 220px，超出滚动）
- 未认领需求归入"未认领"分组
- 按任务总数降序排列

---

## 四、今日收集（🗒）

### 4.1 看板视图
- 四列看板：待跟进 / 跟进中 / 已确认 / 已关闭
- 已转需求且关联需求状态为"已上线"的自动归入已关闭列
- 卡片展示：来源标签、优先级、标题、关联需求状态、客户ID、描述摘要、跟进备注、反馈人、时间

### 4.2 快速录入
- 顶部快速添加栏：输入标题 + 选来源，回车或点击快速添加
- 支持详细填写弹窗（标题、来源、优先级、反馈人、客户ID、提出时间、描述、跟进备注、状态）

### 4.3 状态流转
- 待跟进 → 开始跟进 → 跟进中
- 跟进中 → 确认 → 已确认
- 已确认 → 转入需求（自动预填需求表单，关联反馈记录）

### 4.4 数据管理
- 编辑、删除反馈记录
- 刷新按钮手动重新加载

---

## 五、需求新增 / 编辑

### 5.1 表单字段
| 字段 | 说明 |
|------|------|
| 需求编号 | 自动生成（URQ-N），只读不可编辑 |
| 需求类型 | 用户需求 / 研发需求 |
| 需求名称 | 必填 |
| 需求描述 | 支持 Markdown，可上传 .md 文件，支持编辑/预览切换 |
| 所属计划 | 下拉选择，支持新增计划、删除自定义计划 |
| 优先级 | 紧急🔴 / 高🟠 / 中🟡 / 低🟢 |
| 开发状态 | 待开始 / 开发中 / 联调中 / 测试中 / 已上线 / 已暂缓 |
| 开发进度 | 0-100 数字 |
| 需求认领人 | 文本输入 |
| 预计上线日期 | 日期选择 |
| 提出时间 | 日期选择，新增时默认当前日期 |
| 是否需要设计 | 待设计 / 需要设计 / 不需要设计 / 设计完成 |
| 评审是否通过 | 待确认 / 已通过 / 已拒绝 / 暂缓 |
| 备注 | 多行文本 |

### 5.2 保存逻辑
- 乐观更新（先更新本地数据立即渲染，后台同步飞书）
- 保存后后台静默刷新真实数据

---

## 六、需求详情

- 弹窗展示所有字段
- 需求描述支持 Markdown 渲染
- 底部"编辑"按钮直接进入编辑模式

---

## 七、编号管理

- 新增需求自动生成下一个 URQ-N 编号
- 顶部"🔢 重新编号"按钮：按提出时间顺序重新分配所有编号，补全空缺、修正重复

---

## 八、数据导入

- 从飞书多维表格复制数据粘贴导入
- 支持预览（最多展示20条）
- 导入前确认，覆盖现有全部数据
- 字段顺序：需求编号 · 需求名称 · 需求描述 · 需求类型 · 所属计划 · 优先级 · 开发状态 · 开发进度 · 需求认领人 · 预计上线日期 · 是否需要设计 · 评审是否通过 · 备注

---

## 九、计划管理

- 默认计划：SAE2.3.3、1688商贸版（不可删除）
- 支持在新增/编辑需求时动态新增自定义计划
- 支持删除自定义计划
- 计划列表存储在 localStorage

---

## 十、成员管理

- 默认成员列表内置10人
- 支持在认领弹窗内新增成员
- 支持软删除成员（从列表移除，存 localStorage）
- 成员数据同时从已有需求认领人字段中自动提取

---

## 十一、数据存储

- 后端：飞书多维表格（通过 `/api/feishu` 接口读写）
- 需求数据和今日收集数据分表存储
- 计划列表、成员列表存储在浏览器 localStorage

---

## 十二、其他交互

- 切换到今日收集时隐藏筛选工具栏
- 从今日收集切回需求列表时自动重置所有筛选条件
- 接口加载中显示 loading 状态，失败显示错误信息
- 顶部显示最后更新时间
- 刷新按钮手动重新加载全部数据
